# Sales Analytics Dashboard — Power BI

![Dashboard](dashboard.png)

## Dataset
- Online Retail Dataset (2011)
- 541,909 rows | 8 columns
- Source: Kaggle

## Visuals Built
- KPI Cards (Revenue, Orders, Customers, AOV)
- Bar Chart — Top Products by Revenue
- Bar Chart — Revenue by Country
- Line Chart — Revenue by Quarter
- Slicer — Country Filter (Interactive)

## Key Insights
- UK drives 84% of total revenue
- Revenue grew 117% from Q1 to Q4 — holiday season effect
- DOTCOM POSTAGE is #1 revenue product at 206K

## Tools Used
Power BI Desktop | DAX | Data Modeling
